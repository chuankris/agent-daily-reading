# Agent Daily Reading

目标：把适合你当前阶段的高质量资料，整理成适合手机碎片阅读的中文精读笔记。

使用方式：

1. 白天碎片时间打开 `docs/` 里的文档。
2. 每次只读 1 篇，不追求当天读完。
3. 先看 `为什么读` 和 `记住这 3 点`。
4. 晚上如果有空，再把 `今晚可动手` 做成一个小实验。

当前规则：

1. 优先官方文档、协议规范、经典论文、一线工程团队文章
2. 每次只推 1 到 2 篇，避免信息洪水
3. 先补概念认知，再逐步进入工程实战

目录说明：

- `docs/00-reading-map.md`：阅读地图
- `docs/01-*.md`：首批概念型材料
- `daily/`：后续每天自动新增的阅读材料
- `news/`：后续每天自动新增的行业雷达（模型/工具/路线/开源）
- `references/user-roadmaps/`：你当前这轮转型过程中形成的路线文档归档
- `templates/daily-reading-template.md`：后续每天新增材料的模板
