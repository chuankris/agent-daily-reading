# Daily

这里放后续自动新增的每日阅读材料。

命名规则：

- `YYYY-MM-DD-topic.md`

和 `docs/` 的区别：

1. `docs/`
- 放稳定的核心概念材料

2. `daily/`
- 放每天新增、贴近当前学习阶段的精读或新资料摘要

建议阅读方式：

1. 平时优先看 `docs/` 打基础
2. 碎片时间穿插看 `daily/` 跟进新内容
3. 每篇先读“原文翻译”，再读“中文精读”
